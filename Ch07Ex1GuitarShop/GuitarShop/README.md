# Ch07Ex1GuitarShop

This project is the Guitar Shop app updated for the chapter 7 exercise. I added a Contact Us page in the main Home controller, made a new Razor view for it, and added the nav link in the main layout so it shows as active when that page is open.

The Contact Us page uses a dictionary model and loops through the values in a table. I also added the client-side validation script section to the admin Category AddUpdate view like the Product view uses.
